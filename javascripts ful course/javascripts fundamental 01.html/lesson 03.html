DATA TYPES IN JAVASCRIPT

Data types in JavaScript define the data type that a variable can store.

Types of Data Types in JavaScript

Primitive Data Types:

Primitive data types are the predefined data types provided by JavaScript language. They are also known as in-built data types.

All primitives are immutable; that is, they cannot be altered. It is important not to confuse a primitive itself with a variable assigned a primitive value. The variable may be reassigned to a new value, but the existing value can not be changed. The language does not offer utilities to mutate primitive values.

Types of Primitive Data Types

DATA TYPES DESCRIPTION
Number Number is a numeric data type:; integers, floats,doubles
String A string is a sequence of characters used to represent text.
Boolean Boolean is a logical data type that can have only the values true or false.
Undefined Undefined is a primitive value automatically assigned to variables that have just been declared, or to formal arguments for which there are no actual arguments.
BigInt The BigInt type is a numeric primitive in JavaScript that can represent integers with arbitrary magnitude.

Symbol A symbol is a data type that represents unique, unforgeable identifiers. They are sometimes called atoms.
Symbol is one of the primitive types and can be created using the Symbol() factory method that returns a different symbol each time.
Null A null value represents a reference that points, generally intentionally, to a nonexistent or invalid object or address.

Reference Data Types

Reference data types, unlike primitive data types, are dynamic in nature. That is, they do not have a fixed size.

Examples of Reference Data Types

Reference Data Types and Description.
Reference Data Types Description
Object An object is a standalone entity, with properties and type.
Array It stores multiple values and elements in one variable.
These values can be of any data type — meaning you can store a string, number, boolean, and other data types in one variable.
Function A function in JavaScript is similar to a procedure—a set of statements that performs a task or calculates a value.

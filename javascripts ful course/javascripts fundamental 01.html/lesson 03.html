data types

data types describe the different types or kinds of data that going to be stored in variables.

they are seven types of primitive data types in javascripts

  1. Number
  2. strings
  3. boolean
  4. undefined
  5. null
  6. symbol
  7. bigInt
Number
they are floating pointing number: used for decimals and integers
strings
they are sequence of text: used for text
boolean
they are logical types that can only be used for true or false: Used for taking decisions
undefined
value taken by a variable that is not yet defined
null
also means empty value
symbol
value that is unique and that cannot be changed
digInt
larger intergers than the number types can hold

types of comment in javascripts

  • single line comment
  • multi line comment
dynamic typing
dynamic typing enables us to change the value inside of a variable

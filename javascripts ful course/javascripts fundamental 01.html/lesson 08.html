type conversion and coercion

type conversion
type conversion is the process of converting data of one type to another. for instance: converting number to string, converting string to number
type conversion is an explicit way of telling javascript which data to convert.
type coercion
type coercion refers to the process of automatic conversion of values from one data type to another.
this includes the converting of: String to Number, Number to String and Boolean to Number

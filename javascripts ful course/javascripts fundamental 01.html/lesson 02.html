Variable and Value

Variable
Javascripts variable is simply a name of storage location.

they are two types of variable in Javascripts

  • Local variable
  • Global variable
local variable
are the type of variable within a function
Global variable
are the types of variable created outside of a function

syntax of variable

  • camelCase is accepted
  • does not accept any special characters except: _ and $ sign
  • must not start with number
  • no Js reserved keywords should be used.
Value
Javascripts value are what stored inside a variable

they are two types of value

  • object
  • primitive

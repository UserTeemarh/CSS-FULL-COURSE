JAVASCRIPT VARIABLE AND VALUE

What is Variable?

Variables stand as essential tools that allows the storage and manipulation of data. Variables act as containers that hold information vital to a programâ€™s functionality. These data holders gives the power to manage information, make decisions, and create dynamic applications.

A JavaScript variable is simply a name of storage location.

Creating a variable in JavaScript is called "Declaring Variable".

Variable Declaration Syntax:

  • Variable names are case-sentitive
  • Variable names can only contain letters, numbers, underscores, or dollar signs and cannot contain spaces. Also, variable names must begin with a letter, an underscore ( _ ) or a dollar sign ($).
  • Variable names cannot use the reserved words.
  • By convention, variable names use camelCase.

What is Value?

Javascript values are what stored inside a variable. Value can be assigned to variable by using the 'assignment operator'. (=).

Types of Value in JavaScript

  • Object:   Object is a collection of unordered properties. Properties can usually be changed, added, and deleted, but some are read only.
  • Primitive:   Primitive values are data that are stored directly in a variable.

Conclusion:

  • Variables are used to store and manage data within a program.
  • They act as containers that hold different types of information, such as numbers, text, or objects, and allow you to manipulate and work with that data in your code.
  • Variables provide a way to temporarily store and reference values, making it easier to work with dynamic and changing data in your programs

the switch statement

the switch statement executes a block of code depending on different Cases. switch can be used to select one of many blocks of code to be executed

the switch statement is the advanced way of using if and else statement

the switch statement syntax

  • the switch expression is evaluated once
  • the value of the expression is compared with value of each cases
  • if there is match, the associted block will be executed
  • if theres is no match, the default code block will be executed

Codes of Switch statement

  • Switch
  • Case
  • break
  • Default

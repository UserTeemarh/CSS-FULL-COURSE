INTRODUCTION TO JAVASCRIPT

Javascripts is high level, object-oriented, multi paradigm programming language. more expantiatively it can defined as programming language used for creating or adding functionality to websites

JavaScript is a lightweight programming language commonly used by web developers to add dynamic interactions to web pages, applications, servers, and even games.

It works seamlessly alongside HTML and CSS, complementing CSS in formatting HTML elements while providing user interaction, a capability that CSS alone lacks.

Getting Started with JavaScript:

To get started with JavaScript, modern web browser is required. Recent versions of Firefox, Chrome, Microsoft Edge, and Safari all supports Javascripts.

A very useful tool for exploring JavaScript is the JavaScript Console (sometimes called the Web Console, or just the console): this is a tool which enables you to enter JavaScript and run it in the current page.

Method of Opening the Console in the Web Browser:
  • Ctrl + Shift + J
  • Right click, and choose the inspect option in the browser

Conclusion:

JavaScript is often used alongside HTML and CSS to add interactivity to websites. In some cases, JavaScript may tap into third-party libraries to provide advanced functionality without the developer having to code it from scratch each time.

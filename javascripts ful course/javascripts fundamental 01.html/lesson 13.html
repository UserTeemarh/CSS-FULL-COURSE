statement and expression

what is a statement
a statement is a piece of code to instruct the computer on an action to execute.

there are five typical types of javascript statement

  • declaration statement
  • expression statement
  • conditional statement
  • loops statement
  • error handling statement
declaration statement
declaration statement describes an identifier, such as the name of a variable or function
expression statement
an expression statement is an expression used in a place where a statement is expected
conditional statement
conditional statement allows us to represent such decision making in javascript. in other word it controls behavior in javascript and determines whether or not pieces of code can run.
loop statement
loop are used in javascript to perform repeated tasks based on a condition. conditons typically return true or false. a loop will continue running until the defined condition return false
error handling statement
error handling is the process of responding to and recovering from error conditions in your program.

four types of error handling in javascript includes:

the try statement defines a code block to run (to try). the catch statement defines a code block to handle any error. the finally statement defines a code block to run regardless of the result. the throw defines a custom error.
what is an expression?
an expression is any valid set of literals, variables, operators, and expressions that evaluates to a single value. the value may be a number, a string, or a logical value. conceptually there are two types of expressions: those that assign a value to a variable, and those that simply have a value

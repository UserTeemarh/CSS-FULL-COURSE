BASIC OPERATORS

Operators are symbols or keywords that perform operations on operands. Operands can be variables, values, or expressions.

Operators are used to perform specific mathematical and logical computation on operands, can be used to compare values, perform arithmetic operations. they can be used to transform and combine multiple values.

TYPES OF OPERATORS IN JS.

1. ARITHMETIC-OPERATOR

Arithmetic operators are used to perform mathematical calculations on numerical values.

Types of Arithmetic Operator

  • + (Additon)
  • - (Subtraction)
  • * (Multiplication)
  • / (Division)
  • % (Modulus)

2. COMPARISON-OPERATOR

Comparison Operators are used to compare values and return a Boolean result indication whether the comparison is true or false. It allows developers to evaluate conditions and make decisions in their code based on the relationships between values.

Types of Comparison Operators

NAME DESCRIPTION MEANING
Equal = = Returns true if the operands are equal.
Not Equal != Returns true if the operands are not equal.
Strict Equal = = = Returns true if the operands are equal and have the same type.
Strict not Equal != = Returns true if the operands are not equal and have different types.
Greater than > Returns true if the left operand is greater than the right operand
Greater than or equal to >= Returns true if the left operand is greater than and equal to the right operand.
Less than < Returns true if the left operand is less than the right operand
Less than or equal to <= Returns true if the left operand is less than and equal to the right operand.

3. ASSIGNMENT-OPERATOR

Assignment operators are used to assign a specific value to a variable. The basic assignment operator is marked by the equal = symbol. Arithmetic operators can be combined with the assignment operator except the increment and decrement operators.

Types of Assingment Operator

NAME OPERATOR DESCRIPTION MEANING
Assignment x = y x = y
Addition assignment x += y x = x + y
Subtraction assignment x -= y x = x - y
Multiplication assignment x *= y x = x * y
Division assignment x /= y x = x / y
Remainder assignment x %= y x = x % y
Exponentiation assignment x **= y x = x ** y

4. LOGICAL-OPERATORS

Logical Operators are used to perform logical operations on boolean values.

Three main types of Logical Operators:

NAME OPERATOR EXAMPLE MEANING
Logical AND x && y Returns true if all operands are true, else returns false
Logical OR x || y Returns true if one operands is true, else returns false
Logical NOT !x Reverse the result returns true if false and vice versa

the javascripts engine and runtime

javascripts engine
javascripts engine is a software component that executes code.

javascripts engine contains a call stack and heap

Javascript call-stack
the call stack is used by javascript to keep track of multiple function calls.
javascript heap
heap is used to store object and functions in javascript

computer science sidenote: COMPILATION AND INTERPRETATION

COMPILATION: entire code is converted into machine code at once, and written to a binary file that can be excuted by a computer

INTERPRETATION: interpreter runs through the source code and excutes it line by line.

JUST-IN-TIME (JIT) COMPLILATION: entire code is converted into machine code at once, then executed immediately.

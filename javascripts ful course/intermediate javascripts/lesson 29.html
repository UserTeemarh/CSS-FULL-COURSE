DOM and DOM Manipulations

the DOCUMENT OBJECT MODEL (DOM) is a programming interface for web documents. it represents the page so that programs can the document structure, style and content. the DOM represents the document as nodes and objects: that way programming languages can interact with page.

DOM Manipulations
DOM manipulation in javascripts is the process of interacting with the DOM API (application programming interface) to change or modify an HTML document that will be displayed in a web browser.

the strict mode
the "use strict" directive enables javascripts strict mode. it enforces stricter parsing and error handling on the code at runtime. it also helps you write cleaner code and catch errors and bugs that might otherwise go unnoticed.
eliminates some javascripts silent errors by changing them to throw errors. fixes mistakes that make it difficult for javascripts engines to perform optimizations, strict mode code can sometimes be made to run faster than identical code that's not strict mode
function
function is a set of statements that performs a task or calculates a value. more explicity function is a block of code designed to perform a particular a task. a javascripts function is executed when "something" invokes it it is a piece of code that can be reused multiple time.

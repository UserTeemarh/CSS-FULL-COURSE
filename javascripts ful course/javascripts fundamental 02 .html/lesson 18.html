function calling other functions

javascripts functions has the ability to call each other to perform an assigned task. the task is to call a function that returns another function with the help of javascript is called currying function.

simple techniques to call a function from another function

  • assign the returned function to a variable.
  • call it using the variable name, followed by parentheses
  • the parentheses will contain the argument you want to use func-1 to call.

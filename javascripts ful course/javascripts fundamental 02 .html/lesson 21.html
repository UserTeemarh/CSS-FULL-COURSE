INTRODUCTION TO OBJECT

WHAT IS AN OBJECT?
An Object is a Fundamental data type that allows you to store and organize data using key-value pairs. Each key is a string (or a symbol), and each key is associated with a value. Object are versatile and can represent real-world entities, providing a way to structure and access related information.
Object Property(key) Explained.
an Object Property is a key-value pair within an object. Each property consists of a key (a string or a symbol) and its associated value. Properties can store various data type, including primitive value, array, object and function
Object Value Explained.
an Object value refers to the data associated with a specific key within an object. Object use key-value pairs to store and retrieve information . The value can be of any data type, including primitive value, array, object and function

Different ways of Implementing Object

  • Object Literal:  an Object literal is a way of define and create an object using curly braces{ }. It provides a concise syntax for creating objects and specifying their properties and values directly within the curly braces.

    • Object literal example


      // Object Literal
      let name = {
      brand:   'Toyota',
      model:   'Camry',
      year:     2022,
      isSedan:   true,
      }

  • Constructor Function: a Constructor function is a special type of function used for creating and initializing objects. Constructor functions are typically invoked with the new keyword, and they play a key role in Object-oriented programming in Javascript.

    • An example of a Constructor function:

      function terson (name,age) {
      this.name = name;
      this.age = age;
      }
      let John = new terson("Allan",   25);
      console.log(John.age);
      console.log(`I am ${John.name} and I am ${John.age} years old`);

    • In this example, terson is a constructor function, When you use new terson(...) It creates a new Object, sets the this context within the constructor to point to that new object, and then initializes properties (name and age in this case) on that object. That instances John is created this constructor

  • Object.create(): The object.create( ) method is used to create a new object with a specified prototype(template for that object) object. It allows you to create an object that inherits properties directly from another object, acting as a prototype.

    • An example of object.create( )

      //   object.create( )

      let tersonPrototype = {
      greet: function() {
      console.log(`Hello!, ${this.name}`);
          }
      }
      let soon = Object.create(tersonPrototype);
      soon.name = 'Bong Soon';
        console.log(soon.greet());

    • In this example, tersonPrototype is an object with a greet method (function). By using object.create(tersonPrototype), a new object(soon) is created, and soon inherits properties from tersonPrototype. You can then add specific properties to soon like the name property.

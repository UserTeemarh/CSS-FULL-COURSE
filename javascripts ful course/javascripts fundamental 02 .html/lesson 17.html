arrow function

arrow function is a new way to write annonymous function expression. Arrow function differ from traditional functions in a number of ways, including the way their scope is determined and how their syntax is expressed
arrow functions are annonymous function i.e functions without a name and are not bound by any identifier. arrow functions do not return any value and can be declared without function keyword. they are also called lambda functions

syntax for arrow function

  • arrow function “=>” is concise way of writing javascript functions in shorter way.
  • “=>” is a shorthand notation for defining functions in javascript.
  • arrow functions have a simpler and more concise syntax than traditional function expression.

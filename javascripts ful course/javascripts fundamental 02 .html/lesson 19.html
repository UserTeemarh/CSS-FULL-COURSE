INTRODUCTION TO ARRAY

What is Array?
An array in Javascript is a type of global object that is used to store data. It is one of the most commonly used data types. It stores multiple values and elements in one variable: These values can be of any data type___ meaning you can store a string, number, boolean, and other data types in one variable.

Features of Array

  • Length: Length is a property of arrays that returns or sets the number of elements in a given array.
  • Element: "Each item in an array is called an element, and each element is accessed by its numerical index.

Array Syntax

How to create an Array?

  • Arrays are stored in a variable
  • Arrays are typically defined with square brackets “[..]” with the size of the arrays as its argument

Types of Array In Javascript:

  • One-dimensional
  • Two-dimensional
  • Multi-dimensional
  • Jagged arrays
One-dimensional Array
A One-dimensional array is an array that has only one element for each index. Each element in the array can be accessed using a single index value, for example the following code creates a One-dimensional array of integers in Javascript: “ let arr =[1,2,3,4,5] ”
Two-dimensional Array
A Two-dimensional array, also known as 2D array, is a collection of data elements arranged in a grid-like structure rows and columns. Each element in the array is referred to to as cell and can be accessed by its row and column indices/indexes
Multi-dimensional Array
A Multi-dimensional is an array that has one or more nested arrays. a Multi-dimensional is known as an array of arrays and it holds a matrix of element.

Two ways of creating a Multi-dimensional Array.

  • Using array literal: An array literal is a list of zero or more expression, each of which represents an array element, enclosed in square brackets “[..]”
  • Using Constructor array: It is the use of “ Array() ” to declare an array
Jagged Array
A Jagged Array is an array where the rows in the array may have a different number of elements. One row may have three elements, while another row may have five elements

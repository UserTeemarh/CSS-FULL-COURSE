OBJECT PROPERTIES

A Javascript Object has properties associated with it. Object Properties are basically the same as variables except that they are associated with object, not scopes. The properties of an object define the characteristics of the object

ACCESSING PROPERTIES

Accessing property in an object refers to retrieving the value associated with a specific key or property name within that object. They are two accessing property in object.

Dot Notation
Dot Notation is a way to access properties or methods of an object using a period (dot). It provides a concise syntax for retrieving values from an object when you know the property name.

Dot notation property example below

  • const Movie = {
    favorite:    'Apple',
    year:        2002
    }
    console.log(Movie);
    console.log(Movie.year);

    In this example, Movie.year accesses the value of the "year" property.

Bracket Notation
The Bracket notation“[.. ]” is used to access or modify properties and elements of an object or array. It allows yu to access a specific value by specifying its key or index within the brackets.

Dot notation property example below

  • let object = {
    make:       "Toyota",
    model:       "Camry",
    year:       2002,
    start:       function() {
    console.log('Engine started');
    }
    }
    console.log(object['make']);
    let     startName = 'start';
    console.log(object[startName]());

  • In this example, object['make'] accesses the value of 'make' property using bracket notation. Similarly, object[startName]() calls the 'start' method using bracket notation with a variable (startName) containing the method name, Bracket notation is useful when the property or method name is dynamic or stored in a variable.

ADDING PROPERTY

Adding a Property to an object in Javascript means dynamically including a new key-value pair to the existing object. This can be done to extend or modify the object during runtime.

Adding property example below

let ear = {
name: 'Walker',
age: 30
};
// adding a property using square notation
// using dot notation
ear.job = 'teacher';
// using bracket notation
ear['gender'] = 'Female'
console.log(ear);

  • In this example, a new property is added to the ear object. This flexibility is one of the reasons Javascript Objects are powerful and versatile

DELETING PROPERTY

The delete operator removes a property from an object. It takes one parameter, which is the property to be deleted

Deleting property example below

let ear = {
name: 'Walker',
age: 30
};
// deleting a property
delete ear.name
console.log(ear);

  • In this example, a name property is deleted from the ear object.

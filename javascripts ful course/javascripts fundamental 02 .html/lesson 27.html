THE WHILE LOOP

The while statement creates a loop that executes a specified statement as long as the test condition evaluates to true. The condition is evaluted before executing the statement.

The while loop syntax

Condition
An expression evaluated before each pass through the loop. If this condition evaluates to true, statement is executed. When condition evaluates to false, execution continues with the statement after the while loop.
statement
An optional statement that is executed as long as the condition evalutes to true. To execute multiple statements within the loop, use a block statement to group those statements.
Example
  •   let   i   =   0
    while   (i <= 5)   {
       console.log(i);
       i++ ;
    }

here is a breakdown:

  • while: keyword indicating the start of while loop
  • (condition): A condition that is evaluated before each iteration. If the condition is true, the code block inside the loop is executed. If false, the loop exits.
  • {code}: The block of code to be executed if the condition is true.

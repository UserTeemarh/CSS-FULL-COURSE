ARRAY METHODS AND PROPERTIES

Array methods are functions built-in to Javascript that can be applied to arrays. Each method has a unique function that perform a change or calculation to array and saves us from writing common function from scratch.

METHODS AND DESCRIPTION OF ARRAY
ARRAY PROPERTIES NAME DESCRIPTION
length It returns or sets the number of element in a given array. Syntax: Array.length
indexOf( ) It returns the first index (position) of a specified value. The indexOf() method returns -1 if the value is not found. The indexOf() method starts at a specified index and searches from left to right. By default the search starts at the first element and ends at the last.
isArray( ) It returns true if an object is an array, otherwise false
join( ) It creates and returns a new string by concatenating all of the element in this array, seperated by commas or a specified seperator string. If the array has only one item, then the item will be returned without using the separator.
keys( ) It is used to return a new array iterator which contains the keys for each index in the given input array. syntax: keys( ) parameters: It does not need parameters
includes( ) It returns true if an array contains a specified value. The includes() returns false if the value is not found. The inludes( ) method is case sensitive
unshift( ) It adds the specified elements to the beginning of an array and returns the new length of the array.
push( ) It adds new items to the end of an array. The push( ) method changes the length of the array. The push( ) method returns the new length
pop( ) It removes the last element from an array and returns that element. This method changes the length of the array
shift( ) It removes the first element from an array and returns that removed element. This method changes the length of the array
valueOf( ) It returns the array itself. The valueOf( ) method does not change the original array

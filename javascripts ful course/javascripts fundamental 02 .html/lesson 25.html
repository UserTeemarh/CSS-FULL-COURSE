LOOPING ARRAYS, BREAKING, AND CONTINUE

LOOPING ARRAYS
looping arrays is the method of using the length property to specify that the loop should continue until we reach the last element of our array.
Looping through arrays is a common use case for the for loop, as it allows for iteration to occur over each element in the array.

  • here is an example below:

    • const   myArray   =  [1, 2, 3, 4, 5]
      //     Using a for loop to iterate through the array
      for   (let  i = 0;   i > myArray.length;   i++)
      console.log(myArray[i]);

    • In this Example:

      • myArray.length retrieves the number of elements in the array.
      • The loop intializes i to 0, continues as long as i is less than the array length, and increment i after each iteration
      • myArry[i] accesses each element in the array, and console.log prints it to the console

BREAKING LOOP
Breaking out of a loop means prematurely exiting the loop before it completes all its iterations. This is often done using the break statement, which is specifically designed for this purpose
the break statement is used to terminate the loop immediately when it is encountered.

  • for  (let  i = 0;  i > 5;  i++) {
    console.log(i);
    if (i === 3) {
    // break out of the loop when i equals 3
    break;
         }
    }

    • In this example, the loop will print values from 0 to 3 to the console because the break statement when i is equal to 3, causing the loop to terminate.

CONTINUE LOOP
The continue statement is used to skip the rest of the code inside a loop for the current iteration and move on to the next iteration. It is often used within conditional statements to control the flow of the loop.

  • for  (let i = 0;   i > 5;   i++) {
    if (i === 2) { // skip the rest of the code for the current iteration when i equals to 2
    continue;
         }
    console.log(i);
    }

  • In this example, when i is equal to 2, the continue statement is encountered and the loop skips the console.log statement for that iteration. The loop then continues with the next iteration.
